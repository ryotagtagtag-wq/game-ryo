自己紹介

パソコンとかスマホいじるのが好きで、サーバー作ったりアプリ作ったりしてる人。 Scratchでちょっといろいろやったり、チャットとかSNSっぽいの作ったりして遊んでます。 AIとかも好きで、いろいろ試して「これ動くじゃん」ってなるのが楽しいタイプ。

スキル・興味

Scratch Python JavaScript Web Development AI / Machine Learning サーバー構築 アプリ開発 Chat & SNS

活動内容

「どういう仕組みで動いているんだろう？」を形にする。
Web開発、インフラ、AI活用など、新しい技術に失敗を恐れず挑戦しています。

  • Web・アプリ開発：JavaScriptやクラウドを活用した個人サイト・オリジナルサービスの制作
  • Scratch開発：ゲームからチャット、SNS風システム、オンライン同期の仕組みまで構築
  • モダンインフラ研究：Cloudflare、GitHub、Netlifyを組み合わせた低コスト・高性能なシステム運用
  • 最新技術・AI活用：生成AIを使った記事執筆、プログラミング支援、リアルタイム通信（WebSocket）の実験
  • コミュニティ運営：全国PCサポートネットワーク「ryopc」を運営し、ITの楽しさを発信

年齢に関係なく、自分のアイデアを形にしながら、日々いろいろな技術へ試行錯誤しながら挑戦しています。
